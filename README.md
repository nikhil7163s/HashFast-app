# HashFast

### Installation

1. **Clone and install dependencies**

```bash
npm install
```

3. **Set up the database**

```bash
# Run migrations
npm run db:migrate

```

4. **Start the development server**

```bash
npm run dev
```

The application will be available at `http://localhost:3000`
